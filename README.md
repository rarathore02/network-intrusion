
# Network Intrusion Detection System
A basic machine learning project that uses the **Random Forest algorithm** to identify network intrusions and classify network activity as normal or malicious.

## Project Overview
This project explores the use of machine learning for detecting suspicious network activity. A Random Forest model is trained on network data to classify whether traffic is normal or intrusive.

## Features
* Data preprocessing
* Random Forest model implementation
* Network intrusion classification
* Result visualization

## Tech Stack
* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* VS Code

# Installation

Clone the repository:
git clone https://github.com/rarathore02/network-intrusion.git

Move into the project folder:
cd network-intrusion

Install dependencies:
pip install -r requirements.txt

# Usage
Run the Python file:
network_classifier.py
